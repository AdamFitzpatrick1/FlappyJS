# FlappyJS

A flappy bird style game made using this tutorial - https://www.youtube.com/watch?v=3SsYZDJdeXk

## Current features
- [x] Ball physics
- [x] Mouse control
- [x] Keyboard control
- [x] Game over message
- [x] Final score
- [x] Random pipe sizes 

## Things to add in the future

* Fancier graphics
* Mutiple levels
* Controller support
